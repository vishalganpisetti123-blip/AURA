# AURA Wardrobe — Stitch AI / Figma AI UI Prompt

Create a premium cross-platform mobile app UI called **AURA Wardrobe** for both **iOS and Android**.

The app is an AI-powered digital wardrobe and personal stylist that helps users scan clothes, organize their wardrobe, generate outfits, manage laundry, and receive AI styling recommendations.

The design should feel like:

* Apple Human Interface Design
* Arc Browser aesthetics
* Pinterest editorial layouts
* Luxury fashion brands
* Minimal Scandinavian design

Avoid:

* Neon colors
* Cyberpunk themes
* RGB lighting
* Gaming-style UI
* Bright gradients
* Overly colorful interfaces

The UI should feel:

* Premium
* Calm
* Elegant
* Fashion-forward
* Intelligent
* Minimal
* Luxury

---

# DESIGN SYSTEM

Use:

* Black & White Monochrome Theme
* Light Mode
* Dark Mode
* Frosted Glass
* Liquid Glass Effects
* Soft Blur
* Thin Borders
* Rounded Corners (20-28px)
* Soft Shadows
* Floating Cards
* High-End Typography
* Editorial Spacing

The ONLY colors in the app should come from:

* Clothing images
* Outfit previews
* Fashion photography

The UI itself remains neutral.

---

# UNIQUE DESIGN LANGUAGE

Introduce a subtle **Dot Matrix Fashion System**.

Use tiny dot-matrix patterns for:

* Loading states
* Section dividers
* Scan animations
* Outfit generation indicators
* Notification badges

Keep it subtle.

Do NOT make it look like Nothing OS.
